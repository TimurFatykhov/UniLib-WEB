# UniLib
UniLib - your personal librarian

## Usage

To install server:



```bash
$ git clone https://github.com/TimurFatykhov/UniLib-WEB.git
$ cd UniLib
$ npm install
```

The next step is to open main page 
- [http://localhost:2001](http://localhost:2001)
